Blog-Post-Layout — Showcase

06.05.2026 · Design-System · Long-Form

Diese Seite demonstriert die Long-Form-Konventionen des Content-Layers: Blog-Posts verwenden 680 px Lesebreite, Inline-Links ohne Underline-Default, JetBrains Mono für Code und einen schmalen vertikalen Trenner statt einer farbigen Border-Stripe für Pull-Quotes.

Heading der zweiten Ebene

Im Fließtext tauchen inline-code-Spans auf dunklem Surface auf. Em-Dashes — wie hier — markieren Reframes. Mittel-Punkt · trennt knappe Aufzählungen.

Ich stehe da, wo Kommunikation auf Technologie trifft – mit Fokus auf Strategie, KI und konkretes Umsetzen.

Code-Block-Beispiel

Blocks brechen die 680-px-Beschränkung und nehmen die volle Spaltenbreite ein:

// Hub-Container
.container {
  max-width: 820px;
  margin: 0 auto;
  padding: 1.5rem;
}

// Card mit Hover-Lift (-6px)
.card:hover {
  transform: translateY(-6px);
  border-color: var(--accent);
}
